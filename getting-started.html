Step 4â€‹

  • First create this REDIRECT_URI env variables value http://localhost:3000/apps/google-calendar/api/callback.

  • Once you get client-id and secret add them to these env variables GOOGLE_CALENDAR_OAUTH_ID, GOOGLE_CALENDAR_OAUTH_SECRET.
